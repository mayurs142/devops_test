### Usage
1. Using docker-compose
```sh
$ git clone git@github.com:mayurs142/devops_test.git
$ cd devops_test
$ docker-compose -f ./docker-compose.yaml up -d
```

2. Using docker swarm on vagrant box
```sh
$ git clone git@github.com:mayurs142/devops_test.git
$ cd devops_test/vagrant_swarm_master
$ vagrant up
```
